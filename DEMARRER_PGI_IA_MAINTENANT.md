## 📋 Instructions simples

### Option 1: Depuis Windows (RECOMMANDÉ)
1. Ouvrez l'Explorateur Windows
2. Allez dans `C:\Users\fvegi\dev\pgi-ia\frontend\`
3. Double-cliquez sur `dashboard.html`
4. Le navigateur s'ouvre avec l'interface PGI-IA

### Option 2: Avec serveur Python
1. Ouvrez PowerShell ou Terminal
2. Tapez:
```powershell
cd C:\Users\fvegi\dev\pgi-ia\frontend
python -m http.server 8000
```
3. Ouvrez votre navigateur à: http://localhost:8000/dashboard.html

### Option 3: Script tout-en-un
J'ai créé un script PowerShell qui lance tout:
`C:\Users\fvegi\dev\pgi-ia\start_pgi_docker.ps1`
