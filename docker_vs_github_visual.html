🎯 Docker vs GitHub - Comprendre la Différence

📚

GitHub = Bibliothèque de Code

C'est quoi ? Un coffre-fort cloud pour ton code source

✅ Sauvegarde : Ton code est safe dans le cloud
📜 Historique : Chaque modification est enregistrée
👥 Collaboration : Travaille avec d'autres devs
🔄 Versions : Retour en arrière possible
# Sauvegarder ton travail git add . git commit -m "Ajout Gemini" git push origin main # Résultat : Code visible sur # github.com/fvegiard/pgi-ia
📦

Docker = Boîte Magique

C'est quoi ? Un conteneur avec ton app + TOUT ce qu'il faut

🏗️ Package complet : App + Python + Libs + Config
🚀 Un clic : docker run = app qui tourne
🌍 Universel : Marche sur Windows, Linux, Mac
🔒 Isolé : Pas de conflits avec le système
# Lancer PGI-IA n'importe où docker run pgi-ia # Résultat : App complète qui # fonctionne sans rien installer

🔗 GitHub + Docker = Workflow Pro

1️⃣ Tu développes

Code sur ton PC
Tests en local
Tout fonctionne ✅

2️⃣ GitHub

git push
Code sauvegardé
Historique conservé 📚

3️⃣ Docker

docker build
Image créée
Prête à déployer 📦

💡 Analogie Simple

GitHub = 📐 Plans IKEA

  • Instructions de montage
  • Liste des pièces
  • Schémas détaillés
  • MAIS tu dois acheter les pièces et monter

Docker = 🏠 Maison Préfabriquée

  • Tout est déjà assemblé
  • Prêt à habiter
  • Livré en un bloc
  • Juste poser et c'est bon !

🎯 Pour PGI-IA
